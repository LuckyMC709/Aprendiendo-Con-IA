# Aprendiendo-Con-IA

Este repositorio contiene un ejemplo sencillo de juego 2D usando Python y pygame.

## Requisitos

- Python 3
- Las dependencias listadas en `requirements.txt`

Puedes instalarlas con:

```bash
pip install -r requirements.txt
```

## Ejecutar el juego

Ejecuta el script principal con:

```bash
python game.py
```

Usa las flechas del teclado o WASD para mover al personaje. Cuando el jugador se acerque al portal se mostrar√° un mensaje.

Presiona ESC o cierra la ventana para salir del juego. Evita al enemigo que patrulla el escenario.
